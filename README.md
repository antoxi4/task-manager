# Task Manager

## Info

This project was created as part of the test task in Ring Labs.
Demo Application located at URL [Demo](http://139.59.215.102:8082)

## Requirements

- Node.js v6.9.2 +

## Installation

- **npm i**

## Run Application

- **npm start**

- [REACT DnD](http://react-dnd.github.io/react-dnd/)
